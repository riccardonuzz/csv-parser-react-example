## Description
This application allow the user to parse CSV, JSON and text file data.

## Installation

    git clone https://github.com/riccardonuzz/csv-parser-react-example.git
    cd csv-parser
    npm i
	npm run watch

## Commands
Run the application in dev mode:

    npm run watch

Run tests:

    npm run test

Run ESLint check:

    npm run lint

Run production build check:

    npm run build
